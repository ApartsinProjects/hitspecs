CONCENTRATION

Immersive Systems & Game Development

מערכות אימרסיביות ופיתוח משחקים

Build immersive games and extended-reality worlds.

Rationale

This concentration trains you to build real-time interactive and immersive software: game engines, 3D graphics, virtual and augmented reality, and the AI, simulation, and cloud backends behind them. It pairs a strong computer-graphics and spatial-computing core with vision, generative, and embodied AI to create worlds that render, react, and scale.
